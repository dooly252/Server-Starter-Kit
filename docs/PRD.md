# LinkHub - 카테고리별 링크 모음 사이트 PRD

## 프로젝트 개요

**프로젝트명**: LinkHub

**목적**: Notion을 CMS로 활용하여 유용한 링크들을 카테고리별로 관리하고 공유하는 웹사이트

**CMS 선택 이유**: Notion API를 활용하여 비개발자도 콘텐츠(링크) 관리 가능. 별도 DB 관리 불필요하며, 직관적인 인터페이스로 빠른 콘텐츠 추가/수정 가능

---

## 주요 기능

1. **카테고리별 링크 조회**: 개발, 디자인, 뉴스, 도구 등 카테고리별로 링크 필터링
2. **링크 카드 UI**: 각 링크의 제목, 설명, 태그를 카드 형태로 시각화
3. **검색 및 필터**: 카테고리/태그 기반 필터링 및 키워드 검색 기능

---

## 기술 스택

- **Frontend**: Next.js 15, TypeScript
- **CMS**: Notion API
- **Styling**: Tailwind CSS, shadcn/ui
- **Icons**: Lucide React
- **환경 설정**: .env.local (Notion API Key, Database ID)

---

## Notion 데이터베이스 구조

| 필드명 | 타입 | 설명 |
|--------|------|------|
| Title | Text | 링크의 제목 |
| URL | URL | 실제 이동할 링크 |
| Category | Select | 카테고리 (개발, 디자인, 뉴스, 도구) |
| Description | Text | 링크 설명 |
| Tags | Multi-select | 태그 (예: React, CSS, AI) |
| CreatedAt | Date | 링크 추가 날짜 |
| Pinned | Checkbox | 상단 고정 여부 |

---

## 화면 구성

- **홈 페이지**: 모든 링크를 카테고리별로 그룹화하여 표시. 상단에 카테고리 필터 버튼
- **검색 페이지**: 키워드 또는 태그로 링크 검색 결과 표시
- **링크 카드**: 제목, 설명, 카테고리, 태그 표시. 클릭 시 새탭에서 링크 오픈

---

## MVP 범위

- Notion API로부터 링크 데이터 조회 및 캐싱
- 카테고리별 필터링
- 반응형 링크 카드 UI (모바일/데스크톱)
- 기본 검색 기능
